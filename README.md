# sports-record-tracker
Sports have always been defined by records — the fastest sprint, the highest jump, the most goals scored. These records not only inspire fans but also set new benchmarks for athletes worldwide. With the rapid growth of Artificial Intelligence (AI) and Cloud Computing, it is now possible to analyze large volumes of historical performance data to track records, monitor athletes’ progress, and even predict who is most likely to break the next big milestone.

This project aims to combine machine learning models with cloud deployment to create a real-time, interactive dashboard that shows:
1. Current record holders in various Olympic sports.
2. Athletes who are statistically closest to breaking these records.
3. Trends and insights in record-breaking across both men’s and women’s sports.
